# Zepto-ecommerce-sql-analysis
SQL analysis of Zepto e-commerce dataset for uncovering sales trends, product-related performances, and pricing insights
## Business Problem
Zepto has been a very quick- commerce grocery delivery platform. The main goal of this project is to analyze product and pricing data to identify trends in product availability, discounting, and pricing strategies.
## Dataset
The dataset has information of products sold on Zepto, which includes:
-Product Name
-Category
- MRP
- Discounted Price
- Discount Percentage
- Availability

  ## Tools Used
  - SQL (MySQL)
  - Excel
  - GitHub
  ## Analysis Performed
  - Number of available products.
  - Average discount percentage.
  - The most expensive products
  - Categories with the highest number of products
  - Products with the highest discounts.
  - Price distribution across categories.
 
  - ## Business Impact
  - Optimization of pricing strategies.
  - Identification of high-performing categories.
  - Improve discount campaigns.
 
  - ## Business Questions
  - How many total products are available in the dataset?
  - Which are the top 10 most expensive products?
  - Which products are having the highest discount percentage?
  - What is the average discount across all products?
  - Which products have no discount applied?
